# Quiet Reminder Homebrew tap

Homebrew tap for [Quiet Reminder](https://github.com/quietapps/QuietReminder) — a native macOS menu bar app that flies a banner across your screen before every meeting.

## Install

```sh
brew tap quietapps/quietreminder
brew install --cask quietreminder
```

## Update

```sh
brew update
brew upgrade --cask quietreminder
```

## Uninstall

```sh
brew uninstall --cask quietreminder
brew untap quietapps/quietreminder
```
